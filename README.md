# 订阅中心原型

一个基于原生 HTML + JavaScript 的会员订阅页原型，可直接在浏览器中打开，也可以通过本地静态服务器预览。

## 项目特点

- 无构建工具，打开即用
- 使用 Tailwind CSS CDN 完成样式
- 使用 Lucide CDN 渲染图标
- 通过 `app.js` 中的套餐配置动态渲染页面

## 目录结构

```text
.
├── app.js
├── index.html
└── README.md
```

## 本地启动

### 方式一：直接打开

这是一个纯静态页面，双击打开 [index.html](/Users/qianwentao/Desktop/会员中心/会员中心/index.html) 即可预览。

适合快速查看页面效果。

### 方式二：使用本地静态服务器（推荐）

在项目目录中执行：

```bash
cd /Users/qianwentao/Desktop/会员中心/会员中心
python3 -m http.server 4173
```

浏览器访问：

```text
http://localhost:4173
```

推荐使用这种方式，后续如果接入接口、图片资源或更多静态文件，兼容性会更稳定。

## 页面说明

当前页面包含以下模块：

- 顶部套餐切换导航
- 会员主卡片展示
- 权益对比表展开/收起
- 时长与计费方案选择
- 升级超级会员引导
- 底部固定购买栏

## 配置入口

主要业务配置位于 [app.js](/Users/qianwentao/Desktop/会员中心/会员中心/app.js) 顶部的 `plans` 数组。

你可以在这里修改：

- 套餐名称
- 描述文案
- 图标
- 渐变主题
- 不同时长价格
- 权益列表

## 开发建议

如果后续准备继续迭代，建议按下面方向逐步升级：

- 将样式与逻辑拆分得更细
- 抽离套餐配置为独立数据文件
- 增加真实购买流程或接口联调
- 迁移到 Vue / React 等框架
- 补充移动端适配与交互细节

## 部署

仓库内已包含 `.vercel` 目录，说明这个项目适合直接作为静态页面部署到 Vercel。

如果只是本地演示，使用 `python3 -m http.server 4173` 即可。
